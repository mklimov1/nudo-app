# Nudo — тренажёр слов

Небольшое приложение для запоминания иностранных слов. Добавляете слова с переводом,
затем в режиме тренировки они попадаются в случайном порядке в виде карточек, а вы
вписываете перевод. Карточка переворачивается по оси X и показывает правильный ответ.

## Стек

- **React + TypeScript + Vite**
- **Ant Design** — UI
- **IndexedDB** (через `idb`) — хранение данных прямо в браузере

## Импорт JSON

На вкладке «Мои слова» кнопкой **«Импорт JSON»** можно загрузить слова из файла
в текущий словарь. Поддерживаются три формата:

**1. Массив объектов** (самый гибкий). Синонимы ключей: `word`/`term` для слова,
`meaning`/`target` для перевода, `comment` для комментария:

```json
[
  { "value": "apple", "translation": "яблоко", "description": "фрукт" },
  { "value": "book", "translation": "книга", "translationDescription": "предмет" },
  { "word": "water", "meaning": "вода" }
]
```

**2. Массив пар-кортежей** `[слово, перевод]`:

```json
[
  ["apple", "яблоко"],
  ["book", "книга"]
]
```

**3. Простой словарь** `{ "слово": "перевод" }`:

```json
{
  "apple": "яблоко",
  "book": "книга"
}
```

Обязательны только слово и перевод, поля с комментариями — необязательные.

## Команды

```bash
npm install    # установка зависимостей
npm run dev    # дев-сервер (http://localhost:5173)
npm run build  # прод-сборка в dist/
```
